# Fact-Checking Web App

An AI-powered fact-checking system that reads PDF documents, extracts factual claims, verifies them against live web data, and flags misinformation.

# Features:-

- Upload PDF files
- Extract factual claims automatically
- Verify claims using live web search
- Detect fake or outdated statistics
- Streamlit frontend
- OpenAI integration
- Tavily web search integration
- Public deployment support


# Tech Stack:-

- Python
- Streamlit
- OpenAI API
- Tavily Search API
- PyPDF2
- Pandas
